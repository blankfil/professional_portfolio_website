# Evans Morara Portfolio Website

This is my personal portfolio and services website built with Django + Tailwind CSS.  
It showcases my professional background, skills, and services in ICT, STEM training, web development, and programming.

- Responsive design with Tailwind CSS for modern UI/UX
- Service cards with hover animations and interactive layouts
- Contact form integrated with Formspree for direct email submissions
- Dark mode support for accessibility and user preference
- Deployed seamlessly via Vercel for fast, global hosting

 Tech Stack
- Backend: Django (Python)
- Frontend: Tailwind CSS
- Forms:Formspree integration
- Deployment: Vercel
